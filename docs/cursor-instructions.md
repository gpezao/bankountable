# Instrucciones para Cursor AI

Tu tarea es implementar el proyecto completo siguiendo el documento:
`docs/plan-implementacion.md`

Prioridades:
1. Código limpio y modular.
2. Implementación fase por fase.
3. No avanzar a una fase sin que la anterior cumpla los criterios de aceptación.
4. Hacer commits pequeños y descriptivos.
5. Documentar todas las decisiones técnicas en `docs/decisiones/`.
6. Todo debe funcionar con ambientes virtuales o docker. Nada con instalacion en el mismo pc.


Puedes crear, modificar o eliminar archivos libremente.
